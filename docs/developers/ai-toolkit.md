Accelerate your Solana mobile development with AI-powered tools. This guide covers Claude Code skills and other AI resources to help you build faster.

## Claude Code Skills

[Claude Code Skills](https://docs.anthropic.com/en/docs/claude-code/skills) provide specialized knowledge and capabilities for specific development tasks. Install these skills to enhance your AI-assisted development workflow.

### Solana Mobile Skills

Skills for building mobile dApps with the Solana Mobile Stack:

- [**install-wallet-ui-react-native**](https://skills.sh/wallet-ui/wallet-ui/install-wallet-ui-react-native) - Set up Mobile Wallet Adapter in React Native using the Wallet UI library

```bash
npx skills add https://github.com/wallet-ui/wallet-ui --skill install-wallet-ui-react-native
```
